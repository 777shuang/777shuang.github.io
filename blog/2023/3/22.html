XP.cssを使えばダサいサイトをなんとかできる話

XP.cssとは、WindowsXP風のウィンドウをデザインできる最強のCSSです
これさえあればこのサイトみたいにダサいページでもなんとかなります
基本的に公式の説明を見とけばいいのですが、TreeViewのところでハマったのでメモ。

説明どおりにTreeViewを組み込むと、▼印が表示されてしまいます。でも、公式の説明ページにはそんな印はない。ソースを読んだところ、CSSで 

summary

{

    user-select: none;

    cursor: pointer;

    display: inline;

}

というのを宣言すれば消える、ということがわかりました
